# 🎬 Movie Reservation System

## 📌 Backend (ASP.NET Core Web API)
- User Authentication (JWT)
- Role-based Authorization
- Movie Management
- Show Scheduling
- Seat Booking System
- Booking Management
- SQL Server + EF Core

- Movie Browsing & Details View

- Booking Flow Pages
- Authentication Pages (Login/Register)
- Admin Dashboard Views
